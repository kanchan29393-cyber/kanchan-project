## Email Spam Detection
This is a Machine Learning project where I built a system to classify emails as Spam or Not Spam.

## About Project
In this project, I worked on email data and applied machine learning techniques to detect spam messages automatically.

## Technologies Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- NLTK  

## How it Works
- First, the dataset is loaded and cleaned  
- Then text is processed and converted into features  
- A machine learning model is trained  
- Finally, the model predicts whether an email is spam or not  

## Result
The model is able to classify emails into spam and non-spam categories.

## Conclusion
Through this project, I learned how machine learning can be applied to solve real-world problems like spam detection.
